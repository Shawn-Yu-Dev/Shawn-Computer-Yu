
Shawn_Yu's Computer System (SYCS)A lightweight terminal-based mock operating system built with C++. It features user authentication, a real-time system clock, and a robust interactive calculator, providing a clean and professional console experience.🚀 Key FeaturesSecure Boot & Authentication: Includes a system boot animation and a user registration process. Password input supports hidden echoing for enhanced security.Real-time System Clock:Displays current system time dynamically (HH:MM:SS).Interactive Controls: Press [Space] to pause/resume the clock and [Q] to return to the main menu instantly.Smart Calculator:Supports standard arithmetic: +, -, *, /.Built-in input validation to filter out illegal characters and prevent crashes.Utilizes long double for high-precision calculations.Low-level Terminal Optimization: Uses termios and fcntl to achieve non-blocking key listening (kbhit), allowing the system to respond to keys without pressing "Enter".🛠 Technical ImplementationLanguage: C++11 or higher.Libraries:<termios.h> & <unistd.h>: Terminal mode control (Raw Mode) for Unix-like systems.<chrono> & <thread>: High-precision timing and system refresh rates.<fcntl>: Non-blocking I/O implementation.Compatibility: Optimized for Unix/Linux-based terminals (including macOS Terminal, WSL, Ubuntu, etc.).🖥 How to RunRequirement: Ensure you have the g++ compiler installed.Compile:bashg++ -o sycs main.cpp
